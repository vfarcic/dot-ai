## Chore: Add Internal Git Utilities (PRD #362, Milestone 1)

Adds shared git operations (`cloneRepo`, `pullRepo`, `pushRepo`) as internal utilities in `src/core/git-utils.ts`, supporting PAT and GitHub App authentication. Refactors `user-prompts-loader.ts` to use these shared utilities instead of inline `execAsync` calls.

These utilities are not yet exposed as MCP tools — that will come in PRD #395 (Git Push Recommend Integration).

**Also includes:** Helm chart support for GitHub App credentials (`secrets.githubApp.*`), path traversal protection, credential scrubbing, and operation timeouts. ([#393](https://github.com/vfarcic/dot-ai/pull/393), [#362](https://github.com/vfarcic/dot-ai/issues/362))
